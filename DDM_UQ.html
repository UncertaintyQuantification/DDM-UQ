Description

Automate and high-throughput uncertainty quantification and estimation in differential dynamic microscopy. It allows 2-D particle movement simulation; and fully automated and high-throughput estimation and analysis of image structure function, intermediate scattering function and mean squared displacement (MSD) using Gaussian process regression. Simulation module takes about 10 seconds, and processing and analysis part take only around 1.5 minutes for 1000 images with 480*480 pixels.

Installation

To use this package, please make sure you installed the optimization toolbox in MATLAB.

Reference

Uncertainty quantification and estimation in differential dynamic microscopy
Gu, M., Luo, Y., He, Y., Helgeson, M. E., & Valentine, M. T. (2021). Uncertainty quantification and estimation in differential dynamic microscopy. Physical Review E, just accepted, arXiv preprint arXiv:2105.01200.

Syntax

simulation(option)
processing(I,option)
analysis(option)
simulation(option) takes a structure variable option to:
  • simulate 2-D particle movement.
  • calculate the numerical MSD.
  • give plots for simulated particle trajectories, and numerical mean squared displacement (MSD) for each particle versus lag times with median MSD.
processing(I, option) takes intensity profiles I and a structure variable option to:
  • transform input lag time (dt) into log space and select 25 points.
  • process Fourier transformation for the differences of intensity profiles on only pre-selected dt (or any user defined set), and calculate corresponding image structure function (Sqt/Dqt).
  • process Fourier transformation for the original intensity profile instead of the differences, which will be used for parameter estimation in analysis module.
analysis(option) takes a structure variable option to:
  • estimate noise variance , and . (See equation (4) in reference paper.)
  • use predictive samples from Gaussian Process Regression (GPR) to robustly estimate the image structure function (Dqt/Sqt), intermediate scattering function (fqt), mean squared displacement (MSD) and other quantities of interest.
  • give plots for predictive Dqt and fqt for different qs and 95% predictive intervals, bias detection plot for Dqt versus q to show whether system is fully decayed or not, unaggregated MSD for different qs and lag times, and aggregate MSD versus lag time with error bar that stands for 95% predictive interval. A line of true MSD for comparsion will be added if the reference value is provided. (For simulated data the generated using simulation module, median of numerical MSD will be considered as true MSD as it close to theoritical value).

Input Arguments

I - intensity profiles

Intensity profiles, specified as an array with size nx*ny*len. nx is number of pixels of a image in x direction, ny is number of pixels in y direction, len is the total number of the stack of images that user wants to analysis.
Data Types: single | double | int8 | int16 | int32 | int64 | uint8 | uint16 | uint32 | uint64

Option - structure variable with many fields

A structure variable with many fields to store parameter values or inputs. For available options of each module and default setting of parameters, please see simulation.m, processing.m and analysis.m files for details.
